========================================
WHAT TO CHECK IF SOMETHING'S WRONG:
========================================

1. NO INITIALIZATION MESSAGE?
   → EA not attached properly
   → AutoTrading disabled
   → Check Experts tab for errors

2. NO RUNTIME MESSAGES?
   → EA is waiting for session time
   → Check current time vs session hours
   → Check if filters are blocking

3. TOO MANY FILTER BLOCKS?
   → Filters may be too strict
   → Consider adjusting filter settings
   → Check market conditions

4. DASHBOARD NOT UPDATING?
   → Check EnableDashboardExport = true
   → Check JSON file exists in MQL4/Files
   → Wait 5-10 seconds for first export

========================================
HOW TO VIEW THE LOG:
========================================

1. Open MT4
2. Click "Experts" tab at bottom
3. Scroll to see all messages
4. Right-click → "Clear" to clear log
5. Right-click → "Save" to save log
